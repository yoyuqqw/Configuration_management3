# Инструмент командной строки для учебного конфигурационного языка

## Описание
Данный инструмент предназначен для обработки текстов, написанных на учебном конфигурационном языке. Он преобразует входной текст в формат JSON и выводит его в стандартный вывод. 

Инструмент проверяет входной текст на синтаксические ошибки и в случае их обнаружения возвращает соответствующие сообщения.

---

## Синтаксис учебного конфигурационного языка

### Комментарии
- **Однострочные:**  
  ```
  C Это однострочный комментарий
  ```
- **Многострочные:**  
  ```
  {#
  Это многострочный
  комментарий
  #}
  ```

### Массивы
Массивы задаются с помощью конструкции:
```
(list значение значение значение ... )
```

### Словари
Словари описываются следующим образом:
```
begin
  имя := значение;
  имя := значение;
  имя := значение;
  ...
end
```

### Имена
Имена переменных должны соответствовать следующему регулярному выражению:  
```
[_a-zA-Z]+
```

### Значения
Значениями могут быть:
- Числа.
- Строки.
- Массивы.
- Словари.

### Строки
Строки заключаются в одинарные кавычки:
```
'Это строка'
```

### Константы
- **Объявление констант:**  
  ```
  имя is значение;
  ```
- **Вычисление константных выражений:**  
  ```
  ${имя 1 +}
  ```
  Результат вычисления — значение.

### Операции и функции
На этапе вычисления поддерживаются:
1. Сложение (`+`).
2. Вычитание (`-`).
3. Умножение (`*`).
4. Деление (`/`).
5. Сортировка (`sort()`).
6. Нахождение минимума (`min()`).

## Установка и запуск

### Установка
1. Клонируйте репозиторий:
   ```bash
   git clone <ссылка_на_репозиторий>
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd <название_проекта>
   ```

### Запуск
Запуск инструмента осуществляется командой:
```bash
./назавание_проекта -i <путь_к_входному_файлу>
```
- `-i` — ключ для указания пути к файлу с входным текстом на учебном конфигурационном языке.

Результат преобразования будет выведен в формате JSON в стандартный вывод.

## Тестирование
Для выполнения тестирования используйте `pytest`
Все конструкции языка покрыты тестами. Для запуска тестов выполните команду:
```bash
python config_tool.py
```
При необходимости, установите `pytest`:
```bash
pip install pytest
```

---
## Структура проекта
- `config_tool.py` - основной файл программы, в котором реализованы все функции для обработки данных.
- `input.txt` - пример входного файла с конфигурацией.
- `output.xml` - выходной файл с результатами преобразования.
